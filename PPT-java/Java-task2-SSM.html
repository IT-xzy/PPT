2.知识剖析

首先，我们要知道SSM框架是什么

这里的SSM框架指的是SpringMVC、Spring和mybatis的三个框架的整合，该框架可完成RESTful风格接口，也是一种比较常用的框架

当然还有另一种SSM框架，即Struts2+Spring+Mybatis的框架

接下来我们将SSM框架拆解，一一解析

相信大家对于mybatis不陌生，这个开源框架由于其极大地便利性，简化了JDBC的操作，而发展起来

而Spring框架由于其IOC的自动注入机制，在提高便利的情况下又提升了安全性

既然他两这么强，那我们肯定是要撮合一下的

但SpringMVC又因其实现REST风格接口横空出世，规范了代码风格

三者如此灵活，自然要搞一出惊世骇俗的“三角恋”
