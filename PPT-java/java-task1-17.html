1.背景介绍

什么是接口

接口是一种用来定义程序的协议，它描述可属于任何类或结构的一组相关行为。在JAVA编程语言中是一个抽象类型，是抽象方法的集合

接口的来源

JAVA中没有多重继承的机制，所以提出了接口的概念，可以完成多重继承的功能
